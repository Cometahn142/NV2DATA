# NV2DATA

A secure DataStore wrapper for Roblox featuring session locking.

세션 잠금을 지원하는 안전한 Roblox용 데이터스토어 래퍼입니다.

## English

### Overview
`NV2DATA` protects player data against duplication glitches.

### Installation
```toml
[dependencies]
NV2DATA = "cometahn142/nv2data@^0.1"
```
